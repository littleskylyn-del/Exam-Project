## Project Description

This project is a responsive Weather Application developed using HTML, CSS, and JavaScript.

The application allows users to search for any city around the world and displays real-time weather information including temperature, weather condition, humidity, wind speed, and a five-day forecast.

The application uses the Open-Meteo API to retrieve weather data. JavaScript Fetch API and async/await were used to handle API requests and dynamically update the webpage.

The interface was designed with responsive layouts to ensure the application works properly on both desktop and mobile devices.


## Technologies Used

 HTML5
 CSS3
 JavaScript
 Fetch API
 Open-Meteo Weather API


## Features

  Search weather by city name
  Real-time weather updates
  Five-day weather forecast
  Weather icons based on conditions
  Error handling
